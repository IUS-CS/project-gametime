# **Contributing**


## **AI Usage**

Our policy for AI usage in this project will be restricted to troubleshooting, and help when necessary. The generated code for help will only be used in the project if the logic and implementation is fully understood.

## **Code Review Guidelines**

For this project, code reviews will only be approved when we are absolutely sure there will be no merge conflicts, and that all the code looks legitimate. 

## **Tools**

The tools we use in this project include IDEs VSCode and PyCharm, along with their built in linters. Other tools we use are Docker, Django, and SQLite. Lastly, we of course use GitHub for our project hosting and version control
